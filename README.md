# Proyek Analisis Data: Bike Sharing Dataset

## Deskripsi Proyek
Proyek ini bertujuan untuk melakukan analisis data pada Bike Sharing Dataset guna memahami pola dan faktor yang memengaruhi jumlah penyewaan sepeda. Proses analisis dilakukan secara end-to-end, mulai dari pengolahan data, eksplorasi data (EDA), visualisasi, hingga penerapan machine learning sederhana sebagai pendukung insight.

## Dataset
Dataset yang digunakan adalah **Bike Sharing Dataset (day.csv)** yang diperoleh dari UCI Machine Learning Repository. Dataset ini berisi data penyewaan sepeda harian dengan berbagai variabel seperti kondisi cuaca, suhu, kelembapan, dan jumlah penyewaan.

## Pertanyaan Bisnis
1. Bagaimana pengaruh kondisi cuaca terhadap jumlah penyewaan sepeda?
2. Apakah terdapat perbedaan jumlah penyewaan sepeda antara hari kerja dan akhir pekan?

## Tools dan Library
